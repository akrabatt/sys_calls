Создает два файла: file1.txt и file2.txt.
Открывает оба файла для чтения и записи.
Использует системный вызов select(), чтобы мониторить оба файла на предмет появления новых данных.
Если в один файл что-то записывается, программа автоматически записывает уведомление в другой файл с текстом: "Data was written to <имя_файла>".
